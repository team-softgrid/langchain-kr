## 이번 분기 반드시 증명할 것
- 오접수율 감소
- 재출동률 감소
- FTFR 상승
- MTTR 단축
- **파일럿 고객 1곳 이상 계약 확정**

## MVP 범위 (절대 확장 금지)
- Top 10 에러코드
- 콜센터 1차 분류 + 현장 코파일럿
- 근거 출처 표시
- 파일럿 사이트 1~2곳

## 역할별 우선순위
- **데이터 담당**: AS 이력 정제, 코드/증상 표준화, 정답셋 관리, 데이터 플라이휠 기반 구축
- **제품 담당**: 질의→추천→조치결과 피드백 플로우 단순화, 신뢰도 스코어 구현
- **운영 담당**: 파일럿 KPI 수집 체계 구축, 주간 리포트, 고객 피드백 수집
- **사업 담당**: 파일럿 계약 조건/ROI 설명자료 확보, LoI/MOU 체결

## 4대 확장축 (파일럿 성공 후)
1. AI 기술콜센터 운영
2. AS 자재 판매 연계
3. 충전소 설비 점검관리 SaaS
4. 지역별 AS 전문가 풀 매칭

## 스타팅 멤버 공통 원칙
1. 기능보다 KPI를 먼저 본다.
2. 근거 없는 답변은 실패로 본다.
3. 현장/콜센터가 실제로 쓰는 흐름으로 설계한다.
4. 주간 단위로 가설-실험-수정 반복한다.
5. **고객 데이터는 해당 고객 소유. 익명화 없이 공용 모델에 반영하지 않는다.**
