# AI 기반 충전기 AS 코파일럿 제안서

> **문서 버전**: v2.0  
> **최종 수정일**: 2026-04-14  
> **용도**: 고객(CPO/제조사) 제출용 정식 제안서  

---

## 1. 제안 배경

### 1.1 시장 환경 변화
전기차 충전 인프라 시장은 '양적 확대'에서 **'운영 품질 경쟁'**으로 전환되고 있습니다.
- 국내 충전기 누적 **약 50만 기** 돌파 (2026.4 기준)
- 정부 보조금 사업자 선정 기준에 콜센터 운영, 유지보수 인력 확보, 고장 데이터 공개 등 **AS 품질 요건 의무화**
- 충전기 유지보수 시장 규모: **연간 8,000억 원 이상**

### 1.2 귀사의 운영 이슈
귀사는 다음 운영 이슈를 상시적으로 경험하고 있습니다.
- 24시간 콜센터 운영 부담
- 오접수 및 불필요 출동
- 진단 미숙으로 인한 재출동
- 보증 AS 장기화로 인한 비용 증가

본 제안은 위 문제를 **RAG 기반 AS 코파일럿**으로 해결해 유지보수 비용과 복구 시간을 동시에 줄이는 것을 목표로 합니다.

## 2. 제안 솔루션
### 2.1 서비스 개요
- 에러코드/증상 입력 시 유사 AS 사례를 검색
- 원인 후보, 조치 단계, 필요 부품을 제시
- 근거 출처를 함께 제공해 신뢰성 확보

### 2.2 확장 서비스 (추가)
- **AS 자재 판매 연계**: 진단 결과 기반 권장 자재 자동 제안/주문
- **지역별 AS 전문가 풀 연결**: 긴급 건에 권역별 전문 인력 자동 매칭
- **AI 기술콜센터(기술지원센터)**: 24/7 1차 기술응대, 접수 품질 표준화
- **충전소 설비 점검관리 SaaS**: 사이트 단위 점검 계획, 이력, 리포트 관리

### 2.3 도입 효과
- 오접수 감소 및 출동 필터링
- 재출동률 감소로 인력 운영 효율화
- FTFR 향상으로 고객 만족도 개선
- 외주 인력 숙련도 편차 완화
- 부품 미지참 출동 감소로 현장 생산성 개선
- 콜센터 1차 해결률 향상 및 야간 대응 안정화

### 2.4 왜 우리 솔루션인가
| 항목 | 우리 솔루션 | 범용 FSM 도입 | 자체 AI 개발 |
|------|:----------:|:------------:|:----------:|
| 도입 비용 | 200~800만/월 | 수억 원 초기비 | 수억 원+ |
| 도입 기간 | **12주** | 6개월 이상 | 12개월 이상 |
| 충전기 AS 특화 | ✅ 에러코드 학습 완료 | ❌ 범용 | △ 처음부터 구축 |
| 즉시 효과 | ✅ 파일럿부터 KPI 측정 | △ 커스터마이징 후 | ❌ 개발 완료 후 |

## 3. 구축 범위
- AS 이력/매뉴얼/장애코드 데이터 연계
- RAG 인덱스 구축 및 주기적 업데이트
- 콜센터/현장용 조회 UI
- KPI 대시보드(오접수, 재출동, FTFR, MTTR)
- 자재 카탈로그/재고 연동(옵션)
- 전문가 풀 매칭 워크플로우(옵션)
- 점검관리 SaaS(일정, 점검표, 점검이력) 모듈(옵션)

## 4. 추진 일정 (12주)
| 주차 | 활동 | 산출물 |
|------|------|--------|
| 1~2주 | 요구사항 확정, 데이터 진단 | 데이터 품질 리포트 |
| 3~6주 | 데이터 정제 및 RAG 구축 | 인덱스 + 데모 환경 |
| 7~9주 | 파일럿 운영(현장/콜센터) | 주간 KPI 리포트 |
| 10~12주 | KPI 검증 및 상용 전환 제안 | 성과 보고서 + 상용 견적 |

## 5. 성과 측정 기준
| KPI | 설명 | 파일럿 목표 |
|-----|------|:----------:|
| 오접수율 | 불필요 출동 비율 | 20% 감소 |
| 재출동률 | 동일 이슈 재방문 | 15% 감소 |
| FTFR | 1회 방문 해결률 | 10% 상승 |
| MTTR | 평균 복구 시간 | 15% 단축 |
| 1차 해결률 | 콜센터 즉시 해결 | 30% 달성 |
| 부품 미지참 재방문율 | 부품 준비 실패 | 20% 감소 |
| 월간 절감액 | 출동 원가 절감 | 정량 측정 |

## 6. 운영/보안 정책
- AI는 권고안 제공, 최종 판단은 엔지니어 수행
- 고객 데이터는 계약 범위 내에서만 사용 (소유권은 귀사에 귀속)
- Private Cloud/On-Premise 옵션 제공 가능
- 계약 해지 시 데이터 완전 삭제 보장
- 개인정보보호법 준수: 개인정보 익명화 처리

## 7. 상용화 모델 (요약)

- 성과연동형 요금제 (옵션)

## 8. 요청 사항
- 파일럿 대상 사이트 지정(1~2곳)
- 최근 12~24개월 AS 이력 제공
- 현장 리더 및 콜센터 담당 각 1명 지정
- 2주 내 상세 설계 워크숍 참석

## 9. 기대 ROI

### 9.1 절감액 계산식
```
절감액 = (감소 출동건수 × 건당 출동비용) + (단축시간 × 시간당 인건비)
```

### 9.2 시뮬레이션 예시
| 항목 | 현재 | 도입 후 (예상) | 절감 |
|------|:----:|:------------:|:----:|
| 월 출동 건수 | 200건 | 160건 (-20%) | **40건** |
| 건당 출동 비용 | 80만 원 | 80만 원 | - |
| 재출동 건수 | 40건 | 28건 (-30%) | **12건** |
| **월 절감액** | - | - | **약 4,160만 원** |
| **연 절감액** | - | - | **약 5억 원** |
| **투자 회수 기간** | - | - | **약 2~3개월** |

> ※ 구독료 대비 5~10배의 비용 절감 효과 예상

## 10. 결론
본 제안은 단순 AI 도입이 아닌, **운영 KPI를 개선하는 비용 절감 프로젝트**입니다.  
파일럿을 통해 정량 성과를 확인한 뒤 단계적으로 확대하는 방식이 가장 안전하고 빠른 상용화 경로입니다.
