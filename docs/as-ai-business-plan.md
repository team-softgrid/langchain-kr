# AI 기반 충전기 AS 코파일럿 사업계획서

> **문서 버전**: v2.0  
> **최종 수정일**: 2026-04-14  
> **작성 목적**: 투자 유치 및 내부 전략 수립  

---

## 1. 사업 개요
- **사업명**: AI 기반 충전기 고장 점검·조치 코파일럿
- **서비스 정의**: 에러코드/증상 입력 시 과거 AS 이력(RAG)을 바탕으로 원인 후보, 조치 절차, 필요 부품을 제안하는 운영 최적화 서비스
- **목표**: 오접수·재출동·중복출동 감소 및 FTFR 향상으로 유지보수 비용 절감
- **핵심 메시지**: AI 기능 판매가 아닌, **운영 KPI 개선과 비용 절감 결과를 판매하는 모델**

## 2. 문제 정의 (Pain Point)
- 24시간 콜센터 의무 운영으로 고정비 부담 증가
- 외주 엔지니어 숙련도 편차로 진단 품질 불균형
- 진단 미숙·오접수로 인한 불필요 출동
- 보증 AS 기간 증가에 따른 제조사 후처리 비용 확대
- 부품 미지참/오진으로 인한 재방문 및 가동률 저하
- 정부 정책이 '양적 확대'에서 **'품질 향상 및 효율적 운영'**으로 전환됨에 따라, O&M 역량이 사업자 선정의 핵심 기준으로 부상

## 3. 시장 규모 (TAM / SAM / SOM)

### 3.1 전기차 충전 인프라 시장 개요
| 지표 | 수치 | 출처/기준 |
|------|------|-----------|
| 국내 충전기 누적 설치 | **약 50만 기** (2026.4 기준) | 환경부/무공해차 통합누리집 |
| 2026년 신규 보급 목표 | 7.1만 기 (급속 4,450 + 중속 2,000 + 완속 65,000) | 정부 보급사업 |
| 충전 인프라 시장 규모 | 2022년 **1.5조 원** → 2030년 **32조 원** (CAGR 45%+) | 업계 분석 종합 |
| 주요 CPO(상위 10사) | 전체 시장의 **약 72%** 점유 | PwC/업계 조사 (2025.9) |
| 정부 보급 예산 (2026년) | **5,457억 원** | 환경부 |

### 3.2 TAM (Total Addressable Market) — 전체 충전기 유지보수 시장
```
충전기 50만 기 × 연평균 유지관리비 120~240만 원/기 = 약 6,000억 ~ 1.2조 원/년
(급속: ~6만 기 × 500만 원 = 3,000억 / 완속: ~44만 기 × 150만 원 = 6,600억)
```
- **TAM ≈ 8,000억 ~ 1조 원/년** (2026년 기준)
- 충전기 보급 확대에 따라 2030년 **1.5조 원 이상** 전망

### 3.3 SAM (Serviceable Addressable Market) — AI 기반 AS 솔루션 대상
- **대상**: 급속 충전기 중심 CPO/O&M 운영사 + 제조사 AS 조직
- 급속 충전기 6만 기 운영 사업자 중 AI 솔루션 도입 의향 사업자 (상위 20개사)
- **SAM ≈ 1,500억 ~ 2,000억 원/년**
  - AI 진단 솔루션 구독: ~500억
  - 기술콜센터 위탁: ~400억
  - 자재 판매 마진: ~300억
  - 점검관리 SaaS: ~300억

### 3.4 SOM (Serviceable Obtainable Market) — 3년 내 확보 목표
- **1차 목표 (12개월)**: CPO/제조사 3곳 × 연 4,000만 원 = **1.2억 원 ARR**
- **2차 목표 (24개월)**: 10곳 × 연 6,000만 원 = **6억 원 ARR**
- **3차 목표 (36개월)**: 20곳 × 연 8,000만 원 = **16억 원 ARR**

## 4. 솔루션 개요
### 4.1 핵심 기능
- **증상 기반 추천**: 에러코드/키워드 입력 시 유사 사례 검색
- **조치 가이드**: 원인 Top 1~3, 권장 조치 순서, 안전 점검 체크리스트 제시
- **부품 제안**: 출동 전 필요 부품 리스트 추천
- **근거 제시**: 출처(파일/시트/행) 기반 답변 제공

### 4.2 운영 시나리오
- **콜센터 1차 분류**: 사용자 실수/자가조치 가능건 선별
- **현장 코파일럿**: 외주 포함 엔지니어의 진단 및 수리 보조
- **품질 환류**: 반복 장애 패턴을 R&D 및 품질팀에 피드백

### 4.3 추가 사업 모듈 (파일럿 성공 후 순차 확장)

|:----:|------|------|:-------------:|
| 1 | AI 기술콜센터 SaaS | 24/7 1차 기술응대 자동화 | 높음 |
| 2 | AS 자재 판매 연계 | 진단 결과 기반 부품 추천/주문 | 중간 |

## 7. 경쟁 분석

### 7.1 경쟁 환경 (Competitive Landscape)
| 유형 | 경쟁자 | 강점 | 약점 | 우리와의 차이 |
|------|--------|------|------|--------------|
| **범용 FSM** | ServiceNow, Salesforce Field Service, MS Dynamics 365 | 글로벌 플랫폼, 대형 고객사 레퍼런스 | 도메인 특화 없음, 도입비 수억 원, 한국 충전기 데이터 미보유 | 우리는 충전기 AS 특화 |
| **산업 AI** | 업스테이지(Solar LLM), 마키나록스 | 강력한 AI 엔진, 범용 산업 적용 | 충전기 도메인 지식 없음, 별도 커스터마이징 필요 | 우리는 사전 학습된 AS 이력 보유 |
| **CSMS 자체 AS** | 각 CPO 자체 관제/AS 시스템 | 기존 운영 체계와 통합 | AI 진단 기능 부재, 숙련도 편차 해소 불가 | 우리는 AI 기반 진단 품질 표준화 |
| **자체 개발** | 대형 CPO/제조사 자체 LLM 구축 | 데이터 보안 완전 통제 | 개발 비용 수억, 12개월+ 소요, 지속 운영 부담 | 우리는 12주 내 도입 + 즉시 운영 |

### 7.2 핵심 차별화 (왜 우리만 할 수 있는가)
1. **도메인 특화 데이터 해자**: 충전기 제조사별 에러코드 정규화 사전 + 수만 건 AS 이력 학습 데이터 보유
2. **즉시 운영 가능한 파이프라인**: RAG 프로토타입이 실데이터로 이미 작동 검증 완료 → 12주 내 파일럿 가능
3. **운영 KPI 기반 성과 설계**: 정확도 자체가 아닌 오접수율·재출동률·FTFR 등 비용 절감을 직접 증명
4. **근거 기반 응답**: 출처(파일/시트/행)를 명시하여 환각 리스크 통제, 엔지니어 신뢰 확보
5. **확장 가능한 플랫폼**: 코어 RAG에서 자재판매·콜센터·점검관리 SaaS로 자연스럽게 확장

### 7.3 진입장벽 구축 전략
- **단기**: 제조사별 에러코드 온톨로지 + AS 이력 학습 데이터 축적
- **중기**: 고객사별 피드백 루프 반영 → 정확도 격차 확대
- **장기**: 멀티모달(사진/소리) 진단 데이터 + 예측정비 모델 → 기술 해자

## 9. 데이터 전략 (Data Moat)

### 9.1 현재 데이터 자산
- 충전기 제조사 AS 이력 데이터: **수만 건** (엑셀 기반, 2021~2024년)
- 에러코드/증상/조치/교체부품 구조화 완료 (JSONL 형태)
- RAG 인덱스 구축 및 검색/응답 파이프라인 작동 확인

### 9.2 데이터 플라이휠 (Flywheel)
```
고객 사용 → AS 이력 자동 축적 → 모델 정확도 향상 → 더 높은 KPI 개선
    ↑                                                          ↓
    ← ← ← ← ← 더 많은 고객 유입 ← ← ← ← ← 성과 기반 레퍼런스 ←
```

### 9.3 데이터 확보 목표
| 마일스톤 | 시기 | 데이터 규모 | 기대 효과 |
|---------|------|:----------:|----------|
| 파일럿 시작 | 0개월 | 3만 건 | 기본 추천 품질 확보 |
| 파일럿 종료 | 3개월 | 10만 건 | KPI 개선 증명 |
| 상용 1년차 | 12개월 | 50만 건 | 제조사 교차 패턴 분석 가능 |
| 상용 2년차 | 24개월 | 200만 건 | 예측 정비 모델 학습 가능 |

### 9.4 데이터 정책 및 법적 프레임워크
- **소유권**: 고객 데이터는 해당 고객 전용. 계약서에 데이터 소유권/사용 범위 명시
- **익명화**: 고객 개인정보(이름, 연락처, 위치) 제거 후 모델 학습 반영
- **공용 모델**: 익명화된 패턴 데이터만 범용 모델에 반영 (고객 사전 동의 필수)
- **삭제 권한**: 계약 해지 시 해당 고객 데이터 완전 삭제 보장
- **개인정보보호법 준수**: 개인정보 처리 방침 수립, DPO(개인정보보호책임자) 지정

### 9.5 IP 전략
- 충전기 에러코드 정규화 사전 → 자체 IP로 확정
- 제조사별 특화 온톨로지 → 영업비밀로 관리
- RAG 파이프라인 및 신뢰도 알고리즘 → 특허 출원 검토

## 10. KPI 및 성과지표
| KPI | 설명 | 파일럿 목표 |
|-----|------|:----------:|
| 오접수율 | 불필요한 출동 발생 비율 | 20% 감소 |
| 재출동률 | 동일 이슈 재방문 비율 | 15% 감소 |
